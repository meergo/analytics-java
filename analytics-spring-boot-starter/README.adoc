= Spring Boot starter for Meergo Analytics

This module provides a link:https://projects.spring.io/spring-boot/[Spring Boot]
starter for the Meergo client.

== Usage

For Maven, include this dependency in your `pom.xml`:

```xml
<dependency>
  <groupId>com.meergo.analytics.java</groupId>
  <artifactId>analytics-spring-boot-starter</artifactId>
  <version>${meergo.analytics.version}</version>
</dependency>
```

For Gradle,

```bash
compile 'com.meergo.analytics.java:analytics-spring-boot-starter:+'
```

In your Spring Boot configuration (`application.properties`, environment, or
similar), provide the property `meergo.analytics.writeKey`. The
autoconfiguration will register an `Analytics` bean in the Spring context
ready for use.
